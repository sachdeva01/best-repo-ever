## How to Add More Expenses

If you want to add more one-time expenses:

### Via Web UI (Recommended)
1. Go to **Expense Tracker** page
2. Click **"Add Expense"** tab
3. Select or create category
4. Enter amount and description
5. Check **"This is a recurring expense"**
6. Select frequency:
   - **"Yearly"** for annual expenses
   - **"Every Few Years"** for multi-year (e.g., 5 for every 5 years)
7. Click **"Add Expense"**

### Via Database Script
Create a new migration script similar to `add_onetime_expenses.py`
